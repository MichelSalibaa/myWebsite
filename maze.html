Maze Lab

Generate a maze, explore it, and reveal the solution path.

Choose a grid size and click Generate to begin.

How it works

  • Use the arrow keys on your keyboard to move from start to finish.
  • The maze starts in the upper-left corner and ends in the lower-right corner.
  • Use Generate to build a new maze with the selected size.
  • Solve highlights the shortest path from start to finish.
